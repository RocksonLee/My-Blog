栈

介绍

栈是常用的一种线性数据结构,修改是按照后进先出的原则进行的,又被称为是后进先出( last\ in\ first\ out )表，简称 LIFO 表

可支持操作

bool empty() 判断栈是否为空

void pop() 弹出栈顶元素

int top() 返回栈顶元素

void push() 插入元素于栈顶

int size() 返回栈大小

void clear() 清空栈

复杂度

插入/弹出 O(1)

访问最顶元素 O(1)

手写栈

struct Stack
{
    int val[N],tot;
    Stack() { cl(val,0),tot=0; }
    void clear() { tot=0; }
    int pop() { return tot==0?0:val[tot--]; }
    int top() { return val[tot]; }
    void push(int x) { val[++tot] = x; }
    int size() { return tot; }
    bool empty() {return tot==0; }
}s;

队列

介绍

队列，顾名思义就是一个有顺序性质的数据结构，与栈的实现方式略有不同，修改是按照先进先出的原则进行的，也被称为先进先出(first\ in\ first\ out)表，简称 FIFO 表。

可支持操作

void push() 插入元素于队头

void pop() 弹出队尾元素

int front() 返回队头元素

int back() 返回队尾元素

int size() 返回队列大小

bool empty() 判断队列是否为空

void clear() 清空队列

手写队列

struct Queue
{
    int val[N],l,r;
    Queue() { cl(val,0),l=1,r=0; }
    void push(int x) { val[++r]=val; }
    int pop() { return val[l++]; }
    int front() { return val[l]; }
    int back() { return val[r]; }
    void clear() { l=1,r=0; }
    int size() { return r-l+1; }
    bool empty() { return (r-l+1)==0; }
}q;

树链剖分

介绍

树链剖分用于将树分割成若干条链的形式，以维护树上路径的信息。

具体来说，通过轻重边剖分将整棵树剖分为若干条链(是每个节点只属于一条链)，把树形结构变为线性结构，然后用其他的数据结构(树状数组，SBT，Splay，线段树)维护每一条链上的信息(本质上是一种优化暴力)。

思想

你首先需要知道以下知识：

  • 重子节点(重儿子)：其子节点中子树最大的子节点。如果有多个子树最大的子结点，取其一。如果没有子节点，就无重子节点。

  • 轻子节点(轻儿子)：父亲节点除重子节点外的所有子节点。

  • 重边：父亲节点到重子节点的边。

  • 轻边：父亲节点到其他轻子节点的边。

  • 重链：由多条重边连接而成的路径(链)。

  • 轻链：由多条轻边连接而成的路径(链)。

在树链剖分中，dfs序(dfn)：一般指的是按照重链优先所访问的顺序

所以你既然一次dfs( 大法师 )解决不了，那么你需要两次(

我们在第一次dfs中，找该节点的重子节点，重子节点就是该节点的子节点中子树最大的节点，所以还需要存下节点的子树大小，不仅如此，还需要存下这个节点的父亲和深度(这个之后再说)

第二次dfs中，我们按照重链优先原则进行dfs，这棵树进行剖分，对于每个节点所属的链，我们做出这样安排，将这条重链最浅的节点作为这个节点所属链的标志，那么我们就可以判断这两个节点是否同一条链啦。

上文说到，树链剖分服务于用数据结构维护信息，所以我们记录下这棵树的dfn序(即按照重链优先原则进行dfs的顺序)，以及dfn序所对应的节点。
